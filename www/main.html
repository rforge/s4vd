The s4vd package provides the s4vd algorithm the ssvd algorithm and additional visualization functions. The package depends on the biclust package available on CRAN. Note this is a preliminary version of the package made available for reviewing purposes.



Package source: s4vd_1.0.tar.gz


If you experience problems installing the R-package you can alternatively paste the following R-Code into your R-Console to run the example.


s4vd source code: s4vd.R


example: example.R


lung cancer data set: lung.RData
